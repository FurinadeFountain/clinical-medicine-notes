# 临床医学课程笔记

面向临床执业医师考纲的本科医学课程整合笔记。涵盖基础医学、临床各系统、临床专题速记、实践技能与麻醉专科，含一份历年选择题库与逐题审校。

笔记以 Markdown 为正本，GitHub 直接渲染（含 `$...$` 数学公式与表格）；同时提供一套 LaTeX 排版管线，可一键编译为单一自包含 PDF（全矢量、零位图）。

## 内容结构

```
notes/
├── 01-基础医学/          基础理论合集、生物化学、简明统计学与流行病学
├── 02-临床医学/          外科总论、呼吸、循环、消化、泌尿、女性生殖、血液、
│                         内分泌、精神神经、运动、风湿免疫、儿科、传染病、理化因素
├── 03-临床专题与速记/    106 个诊断公式、系统整合速记、神经病学梳理、卒中精讲
├── 04-实践技能/          各站体格检查、基本操作、心肺复苏评分要点
├── 05-麻醉专科/          临床麻醉学、疼痛诊疗学、危重病医学
└── 99-附录/              选择题库（含逐题疑点审校）、修订日记
```

完整目录见 [`notes/README.md`](notes/README.md)。编译好的 PDF 在 [`dist/`](dist/)。

## 阅读方式

- 直接在 GitHub 上点开任一 `.md` 阅读；数学公式经 GitHub 内置 MathJax 渲染。
- 或下载 `dist/` 内的 PDF（双栏、点线目录、矢量表格）。

## 从 Markdown 构建 PDF

`latex/` 收录排版管线与样式（`preamble.tex` 样式、`figures.tex` 矢量图、`build_all.py` 组装、`normalize.py`/`convert.py` 文本归一化、`tex2md.py` 反向转换器）。需 XeLaTeX（建议 TeX Live + `ctex`/`tabularray` 等）。管线最初按源 Markdown 目录组装，作参考与可复现之用。
